# 微信小程序学习计时器
一款简洁实用的本地学习计时小程序，专为个人学习打卡设计，数据永久保存，无隔日清零。

## 仓库描述
微信小程序学习计时器 | 本地存储、系统精准计时、后台不掉秒、数据永久累计不清零

## 核心功能
- ✅ 学习任务新增、删除管理
- ✅ 基于系统时间精准计时，切后台/锁屏不中断
- ✅ 学习时长永久累计，无自动清零
- ✅ 开始/暂停一键操作
- ✅ 任务完成自动计数
- ✅ 本地数据存储，安全不丢失

## 项目创建步骤
1. 打开**微信开发者工具**，新建小程序项目
2. 填写小程序 AppID，选择**不使用云开发**
3. 将项目内的默认文件全部替换为本项目代码
4. 点击「编译」，即可正常运行使用

## 使用说明
1. 输入任务名称 + 学分，点击添加创建学习任务
2. 点击「开始」启动计时，点击「暂停」停止计时
3. 计时完成后任务自动重置，完成次数+1
4. 所有学习数据永久保存，无隔日清零

## 技术说明
- 纯前端小程序，无服务器、无云开发
- 数据存储：微信本地缓存
- 计时方案：系统时间戳计算，精准不掉秒
